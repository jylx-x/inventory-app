# inventory-app

A fictional inventory management app for Apple.

This project utilizes:
<ul>
  <li>Express</li>
  <li>Mongoose/MongoDB</li>
  <li>Pug</li>
  <li>Bootstrap</li>
</ul>

To see a live demo: https://salty-waters-58408.herokuapp.com/browse
